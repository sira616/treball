Documentació

Apache Tomcat (abans sota el projecte Apache Jakarta) és un contenidor de servlets desenvolupat a l'Apache Software Foundation. Tomcat implementa les especificacions de servlet i de JavaSercer Pages (JSP) de Sun Microsystems, proporcionant en un entorn per al codi Java a executar en cooperació amb un servidor web. Aquest afegeix eines a la configuració que normalment són en format XML. Tomcat inclou el seu propi servidor HTTP, per això també se'l considera un servidor web independent.
